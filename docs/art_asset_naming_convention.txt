ART ASSET LIST AND FILE NAMING CONVENTION - Vertical Endless Archer RPG

1) Asset Organization Goal
- Keep all art assets easy to find.
- Use one consistent naming style.
- Separate characters, enemies, items, UI, and effects.
- Make future scaling and team work easier.

2) Main Asset Categories
- Characters.
- Enemies.
- Bosses.
- Items.
- UI.
- Effects.
- Backgrounds.
- Tilesets.
- Icons.
- Portraits.

3) Suggested Folder Structure
res://art/

  enemies/
  bosses/

  ui/

13) Naming Rules
- Use lowercase only.
- Separate words with underscore.
- Keep names short but clear.
- Include type, character, state, and index if needed.
- Examples: archer_shoot_01.png, boss_guardian_phase2_01.png.

14) Versioning Rule
If an asset is updated, use a version suffix:
- archer_idle_v1.png
- archer_idle_v2.png
- ui_hp_bar_v3.png

15) Export Rule
- Keep PNG for sprites and UI.
- Keep transparent background for characters, enemies, items, and icons.
- Keep background assets without transparency if needed.
- Keep frame size consistent across the same asset type.

16) Recommended MVP Asset Set
- 1 player archer.
- 3 enemy types.
- 1 boss.
- 9 item icons.
- 6 UI elements.
- 3 background images.
- 4 VFX effects.

17) Practical Tip
- Do not mix many art styles.
- Use one pixel size and one lighting style.
- Keep icons, sprites, and UI visually consistent.
