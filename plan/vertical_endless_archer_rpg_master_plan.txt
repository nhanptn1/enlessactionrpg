VERTICAL ENDLESS ARCHER RPG - MASTER PLAN

1) GAME CONCEPT
- Genre: vertical endless action RPG.
- Player: archer fixed at the bottom of the screen.
- Enemies spawn from the top and move downward.
- Every 10 waves, a boss appears.
- The game continues endlessly with scaling difficulty.

2) CORE GAME LOOP
- Start a wave.
- Enemies spawn in patterns.
- Player attacks automatically or by using skills.
- Gain XP from kills.
- Level up and choose upgrades.
- Clear the wave.
- Fight a boss every 10 waves.
- Repeat with harder waves.

3) MAIN SYSTEMS
- Player system.
- Skill system.
- Enemy system.
- Boss system.
- Wave manager.
- XP and level-up system.
- HUD and UI.
- Game over and restart flow.

8) PLAYER DESIGN
- Fixed archer at the bottom.
- Can move left and right only, or stay fixed.
- Basic bow shot is always available.
- 3 to 4 active skills.
- Gains XP and levels up.
- Uses a simple and readable attack style.

9) SKILL SYSTEM
Basic attack:
- Fast.
- Low damage.
- Always available.

Active skills:
- Multishot.
- Piercing Arrow.
- Arrow Rain.
- Trap Shot.

Skill progression:
- Level 1: basic attack.
- Level 3: Multishot.
- Level 5: Piercing Arrow.
- Level 8: Arrow Rain or Trap Shot.

10) ENEMY DESIGN
Normal enemies:
- Slime Scout.
- Goblin Runner.
- Bat Swarm.
- Skeleton Soldier.
- Wolf Beast.

Tank enemies:
- Armored Brute.
- Shield Skeleton.
- Stone Golem.

Elite enemies:
- Stronger versions of normal monsters.
- More HP.
- Faster speed or special ability.

11) BOSS DESIGN
Boss concepts:
- Corrupted Forest Guardian.
- Dark Ranger Commander.
- Fallen Knight.
- Demon Beast.

Boss rules:
- Boss is larger than normal enemies.
- Boss has 2 to 4 attack patterns.
- Boss has telegraphed attacks.
- Boss changes behavior in phases.
- Boss appears every 10 waves.

12) WAVE SYSTEM
- Waves 1 to 3: easy waves.
- Waves 4 to 6: new enemy types.
- Waves 7 to 9: elites and mixed groups.
- Wave 10: boss wave.
- After wave 10: continue with stronger stats and more enemies.

13) XP AND LEVEL-UP SYSTEM
- Enemies drop XP.
- Basic enemies give small XP.
- Elites give medium XP.
- Bosses give large XP.
- XP bar fills toward next level.
- On level-up, show 3 random choices.
- Player picks one upgrade.
- Upgrade applies immediately.

27) PRODUCTION ROADMAP
Phase 1: Core Prototype
- Set portrait screen.
- Add player attack.
- Add one enemy.
- Add health and damage.
- Add enemy spawning.

Phase 2: Wave System
- Add wave manager.
- Add wave numbers.
- Add wave end detection.
- Add enemy scaling.
- Add boss every 10 waves.

Phase 3: Progression
- Add XP system.
- Add level-up choices.
- Add skills and cooldowns.
- Add skill unlocks.

Phase 4: Boss and Polish
- Add boss scene.
- Add attack patterns.
- Add telegraph warnings.
- Add boss HP UI.
- Add victory and defeat flow.

Phase 5: Art and Audio Polish
- Replace placeholder sprites.
- Add item and skill icons.
- Add music and sound effects.
- Add VFX, screen shake, and hit feedback.

Phase 6: Balance and Content
- Tune XP curve.
- Tune wave difficulty.
- Tune skill cooldowns.
- Tune boss attacks.
- Add more enemy and item variety.

28) SUCCESS CRITERIA
- The game is playable from start to boss.
- The player feels progress through level-ups.
- Boss waves are exciting and clear.
- Endless mode keeps getting harder.
- The UI is readable on a vertical screen.

29) BIGGEST RISKS
- Too many features too early.
- Poor readability on a vertical screen.
- Enemy waves becoming repetitive.
- Skills feeling weak or unbalanced.
- Boss fights being unclear or unfair.

30) BEST PRACTICE
- Build one complete loop first.
- Make it fun before adding content.
- Use simple assets until the game works.
- Add polish only after the core loop feels good.
